## 8. 写入硬性规则

### 只允许修改
- `simple_explanation`（仅限上述 20 个 ID）

### 禁止修改
- `original_explanation`
- `question_traditional` / `question_simplified`
- `options_traditional` / `options_simplified`
- `correct_answer`
- `source_page` / `source_file`
- `reference` / `chapter` / `section`
- `id` / `paper`

### 格式硬性要求
- 严格 4 段式：`1. 考点` / `2. 结论 + 解释` / `3. 逐项解释其他选项为什么错` / `4. 记忆口诀`
- 禁止第 5 段
- 第 3 段标题精确匹配 `3. 逐项解释其他选项为什么错`
- 第 4 段标题精确匹配 `4. 记忆口诀`，正文不重复"记忆口诀"
- 简体中文，禁止繁体字
- 禁止教材引用（"教材第X章""1.1.2a"等）
- 禁止内部审计备注（"新增题待复核""先确认答案""audit""spotcheck"等）
- 第 2 段为正确答案解释
- 普通题第 3 段只解释非正确选项
- 组合题第 3 段先判断 i/ii/iii/iv 小项对错，再判断 A/B/C/D 组合
- 每个错误选项说具体错因，禁止"和正确答案不同/概念不同/主体不同/不符合题意"
