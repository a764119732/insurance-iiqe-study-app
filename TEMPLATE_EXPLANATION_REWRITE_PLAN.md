# 模板化 simple_explanation 只读扫描计划

更新时间：2026-05-31

## 1. 扫描口径

本文件只读扫描当前题库中的 `simple_explanation`，用于识别模板化、空泛、学习价值低的解析。扫描结果是工程排队依据，不是直接写入依据。

识别风险包括：

- `新增题待人工复核` 或类似待复核模板。
- 只重复题干或原始解析。
- 只说“这题考教材某节”。
- 其他选项解释空泛。
- 组合题未解释 i/ii/iii/iv。
- 否定题未解释问法。
- “以上各项 / 以上所有 / 以上皆是”逻辑不清。
- 五段式结构不完整。

## 4. 风险分组处理策略

| 分组 | 处理策略 |
|---|---|
| 新增题待人工复核 | 先看 `original_explanation` 是否足够；不足则进入 Do Not Auto 或人工核对。 |
| 只重复原解析 / 旧模板 | 可优先进入小白解析工程，但必须先人工审计。 |
| 只说考教材某节 | 重写时必须补充具体规则、选项排除和快速判断。 |
| 其他选项解释空泛 | 必须逐个选项解释错因，不能用套话。 |
| 组合题未解释 i/ii/iii/iv | 必须逐项判断每个小项成立与否。 |
| 否定题未解释问法 | 必须在第一段或第二段提醒反向问法。 |
| “以上各项”逻辑不清 | 必须说明 A/B/C 是否全对，或为什么全项选项不成立。 |
