## 3. Risky 题型分类

### 3.1 否定题

只读复扫口径：新增题题干包含“不正确 / 错误 / 并非 / 不是 / 不属于 / 不包括 / 不能 / 不得 / 没有 / 无须 / 不应 / 除外 / 不可保”等否定或排除信号。

| 范围 | P1 | P3 | 合计 |
| --- | ---: | ---: | ---: |
| 新增题否定信号 | 265 | 139 | 404 |

首批例子：`P1-316`、`P1-318`、`P1-319`、`P1-321`、`P1-323`、`P1-333`、`P1-339`、`P1-341`、`P1-352`、`P1-356`、`P3-303`、`P3-304`、`P3-305`。

审计重点：

- 确认 `simple_explanation` 是否明确处理“不是 / 不包括 / 没有”等反向问法。
- 不允许只按正确选项套模板解释，否则容易把排除题解释成正向题。

### 3.2 组合题

只读复扫口径：题干或选项出现 `i / ii / iii / iv` 组合判断。

| 范围 | P1 | P3 | 合计 |
| --- | ---: | ---: | ---: |
| 新增题组合信号 | 145 | 53 | 198 |

首批例子：`P1-338`、`P1-342`、`P1-343`、`P1-361`、`P1-364`、`P1-365`、`P1-386`、`P1-423`、`P1-444`、`P1-461`、`P3-301`。

审计重点：

- 逐条核对 i / ii / iii / iv 的对错，不能只解释正确选项字母。
- `simple_explanation` 应说明每个罗马数字项为何纳入或排除。

### 3.3 “以上各项”题

只读复扫口径：选项包含“以上各项 / 以上所有 / 以上皆是 / 以上全部 / 以上均是”等。

| 范围 | P1 | P3 | 合计 |
| --- | ---: | ---: | ---: |
| 新增题“以上各项”信号 | 211 | 104 | 315 |

首批例子：`P1-337`、`P1-338`、`P1-340`、`P1-348`、`P1-352`、`P1-357`、`P1-370`、`P1-374`、`P1-375`、`P1-384`。

审计重点：

- 若答案为“以上各项”，必须确认所有前置选项均正确。
- 若答案不是“以上各项”，必须说明至少一个前置选项错在哪里。
