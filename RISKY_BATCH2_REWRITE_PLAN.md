## 4. 重写标准

下一轮真正重写时，每题 `simple_explanation` 必须使用五段式：

1. `这题考什么：`
2. `为什么正确答案对：`
3. `其他选项为什么不适合：`
4. `记忆口诀：`
5. `遇到类似题怎么快速判断：`

具体规则：

- 否定题要明确题干问的是“不正确 / 不适合 / 并非 / 不是 / 不属于 / 不会”等反向或排除问法。
- 组合题要逐项解释 i / ii / iii / iv 为什么对或错。
- “以上各项 / 以上所有 / 以上皆是”题不能机械说其他选项都错；若答案是全选，要说明前置选项为何都成立；若答案不是全选，要说明哪些前置选项不成立。
- 不编造 PDF 原文。
- 可以基于题干、选项、`correct_answer`、`original_explanation` 和教材概念写大白话解析。
- 不写“其他选项与题干不一致”这类空泛句子，必须说明错在哪里。
