# Golden Explanation Samples

## 样本总原则

黄金样本不是用来机械套格式，而是用来校准解释质量：解析必须让学生真正看懂题目怎么判断，而不是只满足 4 个标题。

所有 `simple_explanation` 必须遵守：

- 必须严格 4 段式：
  1. 考点
  2. 结论 + 解释
  3. 逐项解释其他选项为什么错
  4. 记忆口诀
- 必须使用简体中文。
- 必须讲清题型、判断方向、正确答案成立的原因、其他选项的具体错因。
- 禁止空话、模板话、内部审计话。
- 禁止出现"原始解析"、"PDF 原始解析"、教材编号、"新增题待复核"等用户不可见审计文字。
- 禁止只写"符合题意"、"概念不同"、"多包或少包某个小项"这类不能帮助学生迁移判断的句子。

## P1-553：最高诚信 / 毋须披露 / 反向例外题样本

适用场景：

- 最高诚信原则。
- 披露义务。
- 题干问"毋须披露"、"不用披露"、"除外"、"例外"。

样本要点：

- 第 1 段说明这是"毋须披露"的例外题，不是问哪些重要事实要披露。
- 第 2 段解释正确答案为什么属于毋须披露：常识范围内的事实，一般人和保险人理应知道，不会因投保人没说而误导保险人。
- 第 3 段解释其他选项为什么必须披露：会影响保险人是否承保、如何承保或保费水平。
- 第 4 段给可迁移口诀："常识、保险人已知、减低风险"。

质量标准：

- 反向方向必须在第 1 段说明清楚。
- 正确答案为什么是例外，必须集中在第 2 段。
- 第 3 段不能只说"其他选项不是例外"，必须说明它们为什么影响承保判断。

## P1-244 / P1-245：合约相对性 / 正向定义题样本

适用场景：

- 题干问"哪个法律原则"、"哪个普通法原则"、"哪个概念"。
- 题干描述第三者不是合约一方，因此不用履行合约条款。

样本要点：

- "哪个法律原则 / 哪个普通法原则"通常是正向定义题，不能误判为反向题。
- 合约相对性 = 合约通常只约束签约当事人。
- 第三者不是合约一方，一般不用履行合约条款，也通常不能直接根据该合约主张权利。
- 第 2 段必须把题干描述和"合约的相对性"连接起来。
- 第 3 段必须解释"第三者责任 / 合法性 / 私隐性 / 订约行为能力"等选项具体错在哪里。

质量标准：

- 不得写成"不正确/不属于的选项"。
- 必须说明合约权利义务通常只在当事人之间产生。
- 错误选项要说明具体概念差异，例如第三者责任是第三者可能担责，私隐性是保密，合法性是合约是否符合法律。

## P1-279：组合题 / 代理法 / 责任方向样本

适用场景：

- 题干或选项包含 i / ii / iii / iv。
- 代理法、转承责任、保险代理人、保险经纪、委托人责任方向题。

样本要点：

- 组合题第 3 段标题仍固定为 `3. 逐项解释其他选项为什么错`。
- 第 3 段内部必须先判断 i / ii / iii / iv：
  - i 对/错：说明原因。
  - ii 对/错：说明原因。
  - iii 对/错：说明原因。
  - iv 对/错：说明原因。
- 再判断 A/B/C/D 组合：
  - 正确组合为什么刚好包含正确小项。
  - 错误组合具体多了哪个错误小项、漏了哪个正确小项。
- 不能只写"多包或少包某个小项"。

代理法判断基准：

- 保险人 = 委托人。
- 保险代理人 = 代理人。
- 保险经纪通常代表客户。
- 责任方向要讲清楚：通常是委托人为代理人在授权范围内的行为承担责任，不能倒过来。

质量标准：

- 不能跳过小项判断，直接凭感觉说 A/B/C/D。
- 不能只说"组合不完整"或"多了小项"，必须指出具体多了/漏了哪个小项以及为什么。
- 责任方向要写清角色，不要把保险代理人/保险经纪/保险人责任倒置。

## P1-1291：保险代理人责任 / 反向题 / 绝对化表述样本

适用场景：

- 题干问"不正确"。
- 选项出现"完全不处于"、"完全没有"、"绝不"、"一定不会"等绝对化表述。
- 保险代理人责任、疏忽意见、转承责任。

样本要点：

- 看到"不正确"要在第 1 段说明这是反向题。
- 正确答案是那个说法过头、方向错误或把责任说没了的选项。
- "机会较小"不等于"完全没有风险"。
- 第 2 段集中解释为什么正确答案错：B 错在"完全不处于"太绝对。
- 第 3 段只解释其他非答案选项为什么不选。
- 不要在第 3 段再解释正确答案。

质量标准：

- 第 2 段必须讲清"较小"与"没有"的差别。
- 第 3 段不得出现"正确答案 B"的补充解释。
- 其他选项要写成"A 不选 / C 不选 / D 不选"，说明它们为什么是正确描述或不是本题要找的错误说法。

## Golden Sample Selection Rule

后续大范围修复必须先判断题型，再选择最接近的黄金样本：

| 题型 / 场景 | 优先参考样本 |
|---|---|
| 毋须披露、最高诚信、例外题 | P1-553 |
| 合约相对性、第三者不是合约一方、正向定义题 | P1-244 / P1-245 |
| i/ii/iii/iv 组合题、代理法责任方向 | P1-279 |
| 不正确、绝对化表述、保险代理人责任 | P1-1291 |

如果题目不属于以上场景，也必须沿用这些样本的共同质量标准：先讲题型和方向，再讲正确答案的因果解释，再给具体错因，最后给可迁移口诀。
